# Answers

## 1. How to run

Requirements:
- Node.js
- npm

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

## 2. Stack choice

I chose React for this project because they are simple, lightweight, and efficient for building a persistent CRUD application.

A worse choice would have been a complex microservice architecture because this project is small and does not need unnecessary complexity.

---

## 3. One real edge case

I handled empty note submissions by validating the input before saving it to the database.

The application also includes a theme changer feature to make the notes app feel more minimal and aesthetic for users.
