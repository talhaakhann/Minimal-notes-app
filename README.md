# Minimal Notes App

A simple persistent notes app built with React, Express, and SQLite.

## Features

- Create notes
- Edit notes
- Delete notes
- Persistent storage with LocalStorage
- Theme changer for a minimal and aesthetic experience

## Requirements

- Node.js
- npm

## Installation

```bash
npm install
```

## Run

```bash
npm run dev
```

The app will run locally in your browser.
